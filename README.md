## 📌 Sobre o Projeto
Este sistema foi desenvolvido como parte de um programa de iniciação em tecnologia. O objetivo é permitir que o usuário estime o **consumo mensal de energia (kWh)** de seus aparelhos eletrônicos em suas casas e visualize o custo financeiro aproximado, auxiliando no controle de gastos residenciais.


O programa utiliza **programação estruturada** e **lógica de decisão (`if/elif/else`)** para classificar a eficiência energética do aparelho analisado.

## 🧮 Fórmula utilizada

$$consumoMensal = \frac{potencia \times horas \times 30}{1000}$$

> **Nota:** O custo estimado é calculado com base em uma tarifa fixa de **R$ 0,75/kWh**.
